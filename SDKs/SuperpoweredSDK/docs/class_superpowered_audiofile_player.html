Detailed Description

Thread safety: you can call any method from any thread, just make sure the same stuff is not running somewhere else currently. All methods are non-blocking. Can not be used for offline processing.

Parameters
playingPlay/pause (read-write).
loopCountPlayback count (read-write). If playback reaches to end and loopCount is 0 (default), then playback stops and playing becomes false.
durationSecondsDuration in seconds (read only). Special values: -1 (load error), 0 (nothing is loaded or loading), >0 (loaded successfully)
positionSecondsThe current position as seconds elapsed. Read only.
positionPercentThe current position as a percentage (0.0f to 1.0f). Read only.

void SuperpoweredAudiofilePlayer::open ( const char *  path)

Opening happens on a background thread, with setting durationSeconds to 0. Opening ends when durationSeconds becomes >0 (loaded successfully) or -1 (load error). Use getError() to get the error message.
